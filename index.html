Perfil de Reginaldo

Reginaldo é estudante de Ciência da Computação. Atualmente cursa a matéria Desenvolvimento de interfaces Web na modealidade de Ensino à distância da PUC-Minas. Desenvolveu tal projeto, baseado em bootstrap a fim de demonstrar o conhecimento trabalhado até o presente momento da realização de tal projeto.

Tal projeto teve como objetivo apresentar algumas das ferramentas que podem ser utilizadas para a criação/modelagem de um website. Neste caso, estamos falando de Bootstrap. Utilizando-se do estilo de layout de GRID, é apresentado um website em formato de portifólio, para que sejam observados tais elementos e as habilidades do construtor do site.

Devido à falta de experiência com as ferramentas, o wireframe disponibilizado foi usado como base, para que existisse um padrão a ser avaliado.
